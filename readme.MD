# Reginald and the She Vampires

An Amstrad CPC Remake/reboot of, and sequel to the 1987 game "Mr Weems and the She Vampires" by Dave Moore (c) 2025.

December 2025 Beta Version!

https://typhonsoft.itch.io/reginald

Source code released under the GPL.

Note you need to use Arnauld's updated CPCTelera with the source from https://github.com/Arnaud6128/cpctelera to build successfully

The Intro module uses the standard Development 1.5 branch (using SDCC3.6), whereas the Main module needs the CPCTelera-next branch (using SDCC4.5).
